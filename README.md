# Wordpress_Installer

Este Script es un instalador Automatizado de Sitios Wordpres para servidores basados en DEBIAN

# Versiones testeadas:

* Debian 8
* Debian 9

# Instalacion:

* git clone https://github.com/byteSalgado/Wordpress_Installer
* cd Wordpress_Installer
* chmod 777 *
* ./install.sh
